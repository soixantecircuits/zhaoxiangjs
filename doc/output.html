ZhaoxiangJS

ZhaoxiangJS gives you access to the DSLR currently connected.

Resource Group ¶

Status API Root  ¶

This resources returns the status of the snapbox and the DSLR.

Retrieve the status

Example URI

GET https://tonicdev.io/gabrielstuff/zhaoxiangjs/branches/master/api/status
